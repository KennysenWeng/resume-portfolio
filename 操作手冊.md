# 履歷作品集編輯器操作手冊

## 1. 專案用途

這是一個本機履歷作品集編輯器，可以在網頁上直接輸入履歷內容、即時預覽版面，並將資料儲存成後端檔案。

- 編輯畫面：按右上角「編輯內容」開啟
- 預覽畫面：右側履歷
- 儲存位置：`data/resume.json`
- 網址：`http://localhost:5173`

## 2. 第一次使用

### 安裝 Node.js

請先確認電腦已安裝 Node.js。開啟 PowerShell 後輸入：

```powershell
node -v
npm -v
```

如果兩個指令都有顯示版本號，代表可以執行本專案。

### 啟動系統

進入專案資料夾：

```powershell
cd F:\resume-portfolio
```

啟動後端 server：

```powershell
npm start
```

看到類似以下訊息代表啟動成功：

```text
Resume editor running at http://localhost:5173
```

接著開啟瀏覽器，進入：

```text
http://localhost:5173
```

## 3. 編輯履歷

預設畫面會以履歷預覽為主。需要修改內容時，按右上角：

```text
編輯內容
```

編輯器會從左側滑出。輸入後履歷預覽會即時更新。

可編輯內容包含：

- 基本資料：姓名、職稱、地點、Email、LinkedIn、GitHub 或作品連結
- 自我介紹：摘要與三個重點數字
- 技能：三組技能分類與內容
- 工作經歷：兩筆工作經歷
- 專案作品：三筆專案作品
- 教育與語言：學歷、證照、語言能力

編輯器上方的快速跳轉按鈕可以快速移動到指定編輯區段。修改完成後，可按編輯器右上角的「×」關閉，回到正式預覽畫面。

## 4. 儲存檔案

編輯完成後，請按右上角：

```text
儲存檔案
```

系統會將目前履歷內容寫入：

```text
data/resume.json
```

看到上方狀態顯示「檔案已儲存」代表完成。

注意：左側輸入時會先暫存在瀏覽器中，但正式要帶到其他電腦或備份時，請務必按「儲存檔案」。

## 5. 列印或另存 PDF

按右上角：

```text
列印
```

瀏覽器會開啟列印畫面。若要輸出 PDF，請在印表機選擇：

```text
另存為 PDF
```

列印時系統會自動隱藏上方工具列與編輯器，只輸出履歷內容。

## 6. 切換版面與主題

### 緊湊 / 標準

右上角的「緊湊」可以切換履歷預覽密度。

- 緊湊：內容較密，適合想放更多資料
- 標準：留白較多，閱讀感較舒服

### 色彩主題

左側完成度下方有三個色彩圓點，可以切換履歷主題色。

主題設定會保存在瀏覽器中。

## 7. 到別台電腦使用

請複製整個資料夾：

```text
resume-portfolio
```

不要只複製 `public`，因為後端與儲存資料也在同一個專案資料夾中。

到新電腦後：

1. 安裝 Node.js
2. 開啟 PowerShell
3. 進入專案資料夾
4. 執行 `npm start`
5. 開啟 `http://localhost:5173`

如果已經按過「儲存檔案」，履歷資料會跟著 `data/resume.json` 一起帶到新電腦。

## 8. 關閉系統

啟動 `npm start` 的 PowerShell 視窗不要關閉，網站才會持續運作。

要關閉系統時：

1. 回到執行 `npm start` 的 PowerShell 視窗
2. 按 `Ctrl + C`
3. 若出現確認，輸入 `Y` 後按 Enter

## 9. 常見問題

### 打不開 http://localhost:5173

請確認：

- PowerShell 是否還在執行 `npm start`
- 是否看到 `Resume editor running at http://localhost:5173`
- Node.js 是否已安裝
- 5173 port 是否被其他程式占用

### 按儲存檔案失敗

請確認：

- 是從 `http://localhost:5173` 開啟，不是直接打開 `index.html`
- 後端 server 是否還在執行
- 專案資料夾是否有寫入權限

### 換電腦後資料不見

請確認原電腦是否有按過「儲存檔案」。

資料應該會在：

```text
data/resume.json
```

如果只靠瀏覽器暫存，資料不會自動跟著資料夾搬移。

### 想備份履歷資料

備份整個專案資料夾即可。最重要的是：

```text
data/resume.json
```

## 10. 檔案結構

```text
resume-portfolio/
  public/
    index.html
    styles.css
    script.js
  data/
    resume.json
  server.js
  package.json
  README.md
  操作手冊.md
```

## 11. 建議使用流程

1. 執行 `npm start`
2. 開啟 `http://localhost:5173`
3. 編輯履歷內容
4. 按「儲存檔案」
5. 按「列印」另存 PDF
6. 備份整個 `resume-portfolio` 資料夾
